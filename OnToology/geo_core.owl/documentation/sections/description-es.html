Ontología de Objetos Geográficos: Descripción volver a índice

Como se mencionó anteriormente, la ontología de Objetos Geográficos se basa en la ontología GeoSPARQL y define la clase Objeto Geográfico, que es equivalente a la clase Feature de GeoSPARQL. A través de la clase Objeto Geográfico permite definir aquellos atributos que son comunes para los distintos tipos de objetos que son descritos en el Catálogo de Objetos Geográficos de la Base Topográfica Nacional a escala 1:100.000.

3.1 Ejemplo de uso volver a índice

Con la finalidad de facilitar la comprensión del uso de esta ontología se ejemplificará la representación de una plaza. En primer lugar, como se puede observar en el código de ejemplo, se crea una instancia de la clase Objeto Geográfico, a continuación el nombre de la plaza, su código de identificación y la fecha en que se creó en el catálogo se representan a través de las propiedades de datos dc:title, dc:identifier y dc:created respectivamente. Finalmente, su geometría se representa mediante la propiedad geosparql:hasGeometry. 

	<https://datos.ign.es/recurso/objeto-geografico/8031> a geo_core:ObjetoGeografico ;
	dc:title 		"Plaza España" ;
	dc:identifier		"8031" ;
	dc:created		"2013-03-07"^^xsd:date ;	
	geosparql:hasGeometry	<https://datos.ign.es/recurso/objeto-geografico/8031/geometry> .

Para la representación de la geometría del objeto geográfico se crea una instancia de sf:MultiPolygon que es una de las clases definidas dentro de la ontología GeoSPARQL. Adicionalmente se emplea la propiedad geosparql:asWKT para representar el tipo de codificación de la geometría; finalmente el tipo de dato del valor de la geometría se representa mediante la propiedad geosparql:wktLiteral. 

	<https://datos.ign.es/recurso/objeto-geografico/8031/geometry> a sf:MultiPolygon ;
        geosparql:asWKT  "MULTIPOLYGON (((-5.418825249998759 35.91208060445847, ..., 35.8904970644321)))"^^geosparql:wktLiteral .
