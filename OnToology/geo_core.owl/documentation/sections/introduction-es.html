Introducción volver a índice

La ontología de Objetos Geográficos ha sido generada con la finalidad de representar las abstracciones de un ente del mundo real. Emplea como base la ontología GeoSPARQL que permite la representación datos geoespaciales, es decir aquellos datos que implícita o explícitamente se refieren a una localización relativa a la Tierra. Adicionalmente, en esta ontología se han definido varias propiedades de datos que generalmente son necesarios al momento de representar los datos asociados.
